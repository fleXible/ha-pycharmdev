_Debugger components for PyCharm to be started dynamically as a service._ 

## Installation

1. Click install.
1. Add pycharmdev and set remote debugger host and port.

## Example configuration.yaml

```yaml
pycharmdev:
  debug_host: "hostname"
  debug_port: port
```
